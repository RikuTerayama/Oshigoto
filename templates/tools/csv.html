表を整える

CSVやExcelを、文字化け確認・変換・重複削除・列整理までまとめて処理します。

ファイル内容はサーバーに送信しません。計測・広告のために外部サービスへ通信することがあります。

{% set tool_intro_what = '表データの文字化け確認、CSVとExcelの変換、重複削除、列の整理をまとめて行えます。' %} {% set tool_intro_steps = ['CSVまたはExcelを選ぶ', 'やりたい操作を選ぶ', 'プレビューで確認する', '結果をダウンロードする'] %} {% set tool_intro_notice = '文字コードや区切り文字が合わない場合は、プレビューで確認してから保存してください。' %} {% include 'includes/tool_intro.html' %}

使う順番

  1. ファイルを選ぶCSVまたはExcelをドラッグ＆ドロップします。
  2. 操作を選ぶ文字コード、CSV⇄Excel、重複削除、列の整理から選びます。
  3. プレビューを見る先頭行を確認して、列や文字化けをチェックします。
  4. 保存するCSV、Excel、またはZIPでダウンロードします。

1. ファイルを選ぶ

CSV または Excel ファイルをドラッグ＆ドロップ、または下のボタンで選択してください。（対応: .csv, .xlsx）

📁

クリックまたはドロップでファイルを選択

ファイルを選ぶと、プレビューと操作設定が表示されます。

2. 操作を選ぶ

Shift-JIS は機種依存文字などで文字化けすることがあります。変換後はプレビューで確認してください。

3. プレビューで確認する（先頭50行）

4. ダウンロードする

確認のポイント

Excelで開く前提ならUTF-8（BOMあり）が扱いやすいことがあります。名簿や顧客データなど、個人情報を含む表は保存先にも注意してください。
